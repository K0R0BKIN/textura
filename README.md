# Textura

AI-powered dictionary application in early development.

## Tech stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- shadcn (Base UI)
- Radix Colors
- next-themes

## Getting started

```bash
git clone https://github.com/K0R0BKIN/textura.git
cd textura
pnpm install
pnpm dev
```
