# Textura

AI-powered dictionary application in early development.

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS 4
- shadcn (Base UI)
- Radix Colors
- next-themes

## Getting Started

```bash
# Clone and install
git clone https://github.com/K0R0BKIN/textura.git
cd textura
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
